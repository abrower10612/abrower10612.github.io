Typography

The font I have chosen for this project is Open Sans which belongs to the sans-serif typography group. I've chosen this because it provides a minimalist, simplistic apperaance and helps the page feel more simple and comfortable.

Section Headers:
600 font weight

For my section headers on each page, I'll be using the Open Sans 600 font weight to provide a bolder look that will stand out against the paragraphs. These section headers will also be a little bit lighter than the headings on each page so the heirarchy will be evident.

Paragraphs:
300 font weight

The paragraphs on my page will be done in a 300 font weight so that it's a little bit lighter but still not too light to make it difficult to see with contrast.
